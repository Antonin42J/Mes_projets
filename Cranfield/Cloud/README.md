# Cloud computing

Date : Décembre 2023

Language : Python

Dans le cadre des technologies d'IA et de Big Data, il est important de pouvoir maitriser le cloud computing. Un des principaux fournisseurs de cette technologie avec de grandes capacités est Amazon avec Amazon Web Services (AWS).

Le but de ce projet était de rendre accessible via une interface web des données actualisées sur la qualité de l'air à différents endroits du monde. Actuellement, les scripts ne seront pas fonctionnels car il faudrait les adapter à un nouveau projet AWS.
